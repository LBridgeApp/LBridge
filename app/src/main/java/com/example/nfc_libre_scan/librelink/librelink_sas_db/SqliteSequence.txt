/* sqlite_sequence - это системная таблица SQLite,
 * которая хранит информацию о последнем использованном значении для автоинкрементного столбца
 * в таблице. Эта таблица автоматически создается всякий раз, когда создается таблица
 * с автоинкрементным столбцом в SQLite.
 * Она используется для поддержки автоинкрементного механизма в SQLite,
 * который позволяет генерировать уникальные идентификаторы для новых строк,
 * добавляемых в таблицу. SQLite хранит максимальное значение автоинкрементного столбца
 * для каждой таблицы в таблице sqlite_sequence.
 * Таким образом, её не нужно менять.
 */
