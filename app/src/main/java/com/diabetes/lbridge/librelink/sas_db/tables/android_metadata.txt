/*
 * В SQLite таблица "android_metadata" используется в контексте Android-приложений
 * и содержит метаданные о локализации базы данных.
 * Эта таблица является частью системы управления базами данных SQLite,
 * которая встроена в Android-платформу.
 * Основная цель таблицы "android_metadata" состоит в том,
 * чтобы хранить информацию о текущей локализации базы данных,
 * то есть языковых настройках.
 * Она содержит всего одно поле, называемое "locale",
 * в котором сохраняется информация о текущей локализации базы данных.
 * Это позволяет Android-приложениям легко определить,
 * какой язык или региональные настройки использовать при работе с базой данных.
 * В таблице "android_metadata" обычно содержится одна запись,
 * которая указывает на текущую локализацию базы данных.
 * Например, значение в поле "locale" может быть "en_US" для английского языка
 * и Соединенных Штатов.
 * Это помогает Android-приложениям корректно интерпретировать
 * и отображать данные из базы данных с учетом локализации пользователя.
 * Таким образом, эта таблица не относится к программистам LibreLink, её не нужно менять.
 */
