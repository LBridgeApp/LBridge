How to quickly import databases from apps:

1) Create a new folder named 'Database' in the Desktop directory.
2) Paste commands in PowerShell:

sas.db from LibreLink:
-------------------------------------------------------
adb shell "su -c 'cp \
/data/data/com.freestylelibre.app.ru/files/sas.db \
/storage/emulated/0/Download/sas.db'; exit"; `
adb pull /storage/emulated/0/Download/sas.db `
$env:USERPROFILE\Desktop\Databases\sas.db
-------------------------------------------------------
